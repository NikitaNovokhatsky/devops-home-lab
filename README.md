# DevOps Home Lab

Домашний проект для изучения DevOps-инструментов.

## Технологии

- Python
- Flask

## Запуск

```bash
python app/app.py
```

## Endpoints

| URL | Описание |
|------|----------|
| / | Главная |
| /health | Проверка состояния |
| /hostname | Имя хоста |
| /info | Информация о системе |
